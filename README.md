## Code Box
* 界面：类似一个悬浮窗，提供了一个按钮。
* 内部：提供了一套可调用的API来实现一些操作。可以自定义程序，写入窗口。写入的程序会通过点击按钮触发。
* 应用：1. 点击按钮，显示：hello world!
        2. 点击按钮，进行桌面截图
        3. 点击按钮，设置备忘事项
